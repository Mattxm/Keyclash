# Keyclash

A multiplayer typing game created in Unreal Engine 4.

## Features
* Matchmaking
* Public/Private game lobbies.
* Speedtest
* Quote type

## Dependencies
* Unreal Engine 4.25
* [Advanced Sessions Plugin](https://forums.unrealengine.com/community/community-content-tools-and-tutorials/41043-advanced-sessions-plugin)
